1. Focus on the text input
  2. Hit the "enter" key

Check the console - you will see a click event on Button B and a submit event on the form.

This is because "Button A" has a "type" of "button". Removing that attribute in the source will emit a click event on Button A.

The behavior of emitting a click event on a button with type "submit" when you hit enter in a focused text input appears to only happen on the first button element nearest the text input.
